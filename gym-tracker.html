Import Program

For adding plans from a personal trainer: give Claude the trainer's plan (photo, PDF, whatever) and ask for a "program import" JSON file, then load it here. New exercises are matched by name against your library — if one already exists it's reused, not duplicated; only genuinely new ones get created. Existing plans/sessions/exercises are never touched or replaced by this.

Danger Zone

Cloud sync works when this app is opened through Pete's Apps launcher (it needs a login session). Opened standalone, it runs local-only — use the backup button above regularly in that case.
